# 📝 Gestor de Tarefas (To-Do List) em Java

## 📌 Descrição
Aplicativo de **linha de comando** em Java para gerenciar tarefas: adicionar, listar, concluir, remover e salvar tarefas em arquivo.  

## ⚡ Funcionalidades
- ➕ Adicionar tarefas  
- 📋 Listar tarefas  
- ✅ Concluir tarefas  
- ❌ Remover tarefas  
- 💾 Salvar e carregar tarefas de arquivo
